# 法学引注文献格式转换

一个纯前端静态工具，用于把知网和常见数据库中的 `GB/T 7714-2015` 文献格式转换为法学引注风格脚注。

## 本地运行

```bash
python3 -m http.server 5173
```

然后访问 `http://localhost:5173`。

## 已支持

- 期刊 `[J]`
- 专著 `[M]`
- 学位论文 `[D]`，首版默认按博士学位论文输出
- 知网文献节式顺序，如 `题名[J].作者.刊名,年份(期号)`
- 批量输入、规范化、编号/排序、复制、未识别提示

## 验证

```bash
node --test
node scripts/build-check.js
```

当前实现不依赖后端或外部 API，适合直接部署到 GitHub Pages。
